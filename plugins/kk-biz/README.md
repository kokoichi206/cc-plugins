# kk-biz

**kk** = kokoichi206 / **biz** = 受託・コンサル業務カテゴリ

受託開発・コンサル業務 (非開発タスク) を支援するプラグイン。面談メモの整理、工数/費用見積もり、ヒアリング設計などを扱う。コマンドは `/kk-biz:<skill>` 形式で呼び出す。

> インストール方法はリポジトリルートの [README](../../README.md) を参照。

## スキル一覧

| スキル                     | 説明                                                                                       |
| -------------------------- | ------------------------------------------------------------------------------------------ |
| `/kk-biz:interview-digest` | 面談メモを人ごとに整理し、採否/アサイン案/最短稼働開始日まで含む経営層相談用の叩き台を生成 |
| `/kk-biz:estimate`         | docs/会話/リポジトリから工数 (振れ幅付き) と費用を見積もり、デモ版と本実装を分けて出力     |
| `/kk-biz:hearing-prep`     | 見積もり精度を上げる顧客向け質問を網羅生成し、各質問で何が決まるかを併記                   |

## 使用例

```
# 面談メモを整理して経営層相談用の叩き台を作る
/kk-biz:interview-digest 採用/ ses/面談/

# docs から工数・費用を見積もる (デモ版と本実装を分ける)
/kk-biz:estimate docs/ --with-demo

# 見積もり精度を上げるためのヒアリング質問を作る
/kk-biz:hearing-prep docs/proposal.md
```

## スキルを追加する

1. `plugins/kk-biz/skills/<name>/SKILL.md` を作成
2. この README の「スキル一覧」「使用例」に追記
3. `claude plugin validate .` で検証
